FIFA-Womans(Datos de copas del mundo femeninas)

 ¿De qué trata este proyecto?

Este proyecto contiene un script en Python llamado fifawomans.py, el cual trabaja con datos relacionados al fútbol femenino.
La idea es tener una herramienta simple para mostrar información, analizar datos o probar cosas relacionadas al tema.


Instalación (paso a paso)

Para usar este proyecto en tu computador:

#1. CLONA EL REPOSITORIO: 
    git clone https://github.com/threeprincesses38-sudo/fifa-womans.git
    cd fifa-womans

#2. Crea el entorno virtual(este punto es opcional)
     python -m venv venv
 
*NOTA: para activarlo ocupa estos comandos en 
       windows:venv\Scripts\activate
       Mac/Linux: source venv/bin/activate

#3. Instala las dependencias 
    pip install -r requirements.txt

#4. Ejecuta el script para mostrar los resultados
     python fifawomans.py


API UTILIZADA

La aplicación consulta automáticamente información de países usando:

https://restcountries.com/

No requiere token ni credenciales.
La API entrega:
	•	nombre oficial
	•	población
	•	región
	•	subregión
	•	capital del país

Y la app combina esos datos con métricas futbolísticas.

¿Como descargar los datos?

En la última pestaña puedes:

1. Ver la tabla filtrada
2. Descargar los datos en CSV
3. Guardar resultados para reproducir el análisis después
